# UI / UX 规格

状态：**垂直切片交互基线已锁定**<br>
适用版本：Vertical Slice 0.1<br>
最后更新：2026-07-13

## 1. 体验原则

- UI 帮助玩家理解动作，不替玩家解释谜底。
- 探索时界面尽可能消失；需要确认、阅读或设置时再出现。
- 所有不可逆动作使用按住确认，所有普通调查使用单次确认。
- 恐怖效果不能破坏退出、暂停、可读性和内容安全选项。

## 2. 屏幕清单

| 屏幕 | 必须功能 | 返回行为 |
|---|---|---|
| 标题 | 开始、设置、内容提示、退出 | 退出游戏需普通确认 |
| 内容提示 | 简版主题、详细展开、降低闪烁入口 | 返回标题 |
| 游戏 HUD | 交互提示、文本框、按住进度 | Esc 暂停 |
| 暂停 | 继续、设置、返回标题、退出桌面 | 返回游戏 |
| 设置 | 主音量、全屏、降低闪烁 | 保存并返回来源屏幕 |
| 时钟 | 时、分调整，确认，取消 | 返回探索且保留本次输入 |
| 结尾 | 文本与标题回流 | 自动回标题 |
| 开发面板 | 状态跳转和调试值 | 发布构建不存在 |

## 3. 安全区与排版

- 逻辑画布：640×360。
- UI 安全区：四边各 16 像素。
- 文本框：`x=24, y=246, w=592, h=90`；最多四行。
- 交互提示：角色上方 12 像素或屏幕底部中心；不得遮挡文本框。
- 标题页主要内容位于中央 60% 区域，详细内容提示使用独立面板。
- 字号基线：正文 16 像素、次要信息 12–14 像素、标题 32–48 像素。
- 中文正文行高至少为字号的 1.35 倍。

## 4. 字体

- 标题允许使用定制像素字形或静态图形字标。
- 正文使用经过中文小字号验证的像素或点阵兼容字体，不强行把矢量宋体缩成模糊像素。
- 数字 `0 1 2 7` 必须在时钟界面中清楚区分。
- 字体许可与实际文件进入 `ASSET_CREDITS.md` 后才可发布。

## 5. 文本框状态

| 状态 | 表现 | 输入 |
|---|---|---|
| 出现 | 120ms 淡入，不缩放 | 暂不接受翻页 100ms |
| 打字 | 每字稳定出现，标点短停顿 | E 立即显示本页 |
| 等待 | 右下角小型静态菱形 | E 下一页 |
| 结束 | 80ms 淡出 | 恢复控制 |
| 关键选择 | 显示动作与后果边界 | 按住或 Esc |

不使用快速闪烁光标。玩家立即显示当前页时仍需看到完整的关键选择说明，不能一次按键同时显示并确认下一页。

## 6. 交互提示

- 默认格式：一个小型 `E` 键帽图形加两到四字动作，如“调查”“查看”。
- 不显示物品正式名称，避免提前泄露“关键碎片”。
- 候选改变后提示至少稳定 100ms 再切换，防止站在重叠区域时抖动。
- 惩罚阶段只允许出口门出现提示，其他提示全部隐藏。

## 7. 按住确认

面对记忆和闭眼均使用同一组件：

- 3 秒线性进度；
- 24 像素细环从顶部顺时针闭合；
- 进度 0–70% 使用旧纸色，70–100% 过渡到焦红；
- 松开在 150ms 内平滑回零，不触发失败声音；
- 完成前 0.4 秒减少环境声，增加选择重量；
- 暂停、窗口失焦或输入设备断开时冻结而非完成。

屏幕必须明确写出“松开取消”。

## 8. 时钟界面

- 面板尺寸不超过 320×160，保留背景环境可见。
- 显示 `HH : MM`，当前编辑段使用边框与轻微亮度同时强调。
- 左右切换时/分，上下调整；分钟默认一步一分，按住 400ms 后加速。
- `Shift + 上下` 可十分钟步进，但不在首次提示中强制说明。
- E 确认当前时间，Esc 取消并保留到再次打开。
- 错误只播放低响度齿轮回转；第三次错误后不在界面内显示答案。

## 9. 标题页

第一视觉层：项目字标；第二层：短句“恢复记忆不是胜利”；第三层：菜单。

- 背景使用静态或极慢速房屋主视觉，不做自动播放视频。
- 菜单默认焦点在“开始”，退出位于最下方并与设置分隔。
- 完成任一结尾返回标题后，短句分别临时变为“记忆会回来”或“忘记不是出口”，直到再次开始或退出。

## 10. 暂停与退出

- Esc 在 100ms 内暂停所有游戏、时间线和音频渐变。
- 暂停面板不故意隐藏鼠标、移动选项或制造假按钮。
- 返回标题明确提示“本次进度不会保存”。
- 退出桌面使用一次确认，不用按住三秒。
- 惩罚阶段退出后再次启动从新游戏开始，与文档存档规则一致。

## 11. 无障碍与舒适性

- 降低闪烁：替换所有高对比快速变化；不改变剧情时间。
- 主音量：0–100%，首次默认 70%，不得要求高音量才能解谜。
- 字幕始终开启，因为关键非语言人声仍有语义文本。
- 所有颜色语义同时使用形状或文字。
- 不使用必须快速反应、连打或精确鼠标操作的交互。
- 窗口失焦自动暂停；重新聚焦不直接恢复，等待玩家确认。

## 12. UX 验收

- 新玩家能在 30 秒内开始游戏并理解移动/调查。
- 所有关键文本在 640×360 下无溢出，200% 系统缩放不影响游戏内部布局。
- 同一次按键不会同时关闭文本和触发场景物品。
- 时钟可只用键盘完整操作，错误不会丢失输入。
- 两个按住选择都能在任意进度取消。
- 暂停、失焦和恢复不会让惩罚时间线或结尾跳帧。
- 内容提示可在开始前完整查看，降低闪烁可从同一流程开启。
