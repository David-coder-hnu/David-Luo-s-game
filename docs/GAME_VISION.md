# 游戏愿景

状态：**已接受**<br>
适用范围：垂直切片与完整游戏<br>
最后更新：2026-07-13

> 本文包含核心剧情剧透。

## 一句话愿景

《地狱轮回》是一款 2D 像素心理恐怖游戏：玩家越主动理解一场家庭惨剧，就越清楚自己操控的男人正是惨剧的制造者，而理解本身会召来下一次惩罚。

## 玩家承诺

游戏向玩家承诺：

- 重要细节都有叙事意义；游戏不会靠纯随机变化浪费玩家的观察。
- 推进来自观察、记忆和主动选择，而不是刷取数值或获得超能力。
- 玩家无法替主角洗清罪责，也无法通过隐藏结局取消受害者遭遇过的事实。
- 即使故事中的轮回永远继续，玩家购买或下载的是一段能够完成、能够回味的作品。
- 恐怖主要来自理解、等待和空间回应，而不是无法预警的高音量惊吓。

## 五条体验支柱

### 1. 好奇心逐渐变成恐惧

第一阶段的问题是“这里发生过什么”；第二阶段变成“我是否真的想知道”；最终问题是“明知真相会带来惩罚，我为什么还要继续”。

每条线索必须同时完成两件事：回答一个旧问题，并制造一个更私人、更难回避的新问题。

### 2. 房子是证人，不是随机迷宫

环境变化是房子对记忆和行为的回应。关键变化必须经过编排，并满足至少一项功能：

- 重新解释玩家已经看过的物品；
- 表明房子记得玩家上一轮的行为；
- 把受害者从“剧情背景”还原为生活过的人；
- 提示惩罚正在靠近。

仅仅为了“每次不一样”而进行的随机位移不属于本作的核心价值。

### 3. 知识既存在于角色状态，也存在于玩家脑中

系统可以记录主角见过哪些线索，以保证叙事连贯；但每个主要章节至少要包含一次真正的知识门槛：玩家必须自行联系信息并采取未被任务标记直接指出的行动。

因此，本作采用“剧情旗标 + 玩家推理”的混合模式，而不把自动开门误称为纯知识驱动。

### 4. 惩罚不可逃避，但面对方式属于玩家

玩家不能改写犯罪事实或永久逃离轮回，但可以选择：

- 先调查什么；
- 是否立刻确认某段记忆；
- 面对真相，还是暂时闭上眼睛；
- 以怎样的顺序理解受害者留下的生活痕迹。

不同选择改变情感路径和结尾表达，不制造“主角其实无罪”的奖励结局。

### 5. 受害者首先是人，其次才是谜底

妻子苏岚和女儿秦禾必须拥有独立于主角罪责的生活痕迹：习惯、愿望、关系、恐惧和尝试离开的行动。她们不能只作为主角悔恨的道具出现。

酒精依赖是犯罪发生时的背景，不是免责原因。文本不得暗示“因为喝醉所以不是他的选择”，也不得把自焚包装成赎罪已经完成。

## 情感曲线

| 阶段 | 玩家主要感受 | 设计手段 |
|---|---|---|
| 醒来 | 陌生、克制的好奇 | 安静空间、可辨认的日常物品 |
| 初探 | 不协调、不愿下结论 | 细节矛盾、缺席的人、被遮挡的痕迹 |
| 拼合 | 发现与自己有关 | 物品承认主角身份、声音距离缩短 |
| 预警 | 明知继续有代价 | 可退出的确认动作、减少环境声 |
| 惩罚 | 失去控制但不失去理解 | 脚本化空间压迫、避免复杂战斗 |
| 再醒 | 熟悉感被破坏 | 有意义的环境回应、行为回声 |
| 结束 | 作品结束，轮回未结束 | 两种面对方式、同一不可撤销事实 |

## 叙事事实

- 主角秦峥长期酗酒并实施家庭暴力。
- 妻子苏岚准备带女儿秦禾离开。
- 秦峥在醉酒后杀害妻女，随后纵火并自焚。
- 房子是他死后的惩罚空间，也是被压抑记忆的结构化表现。
- “地狱之神”是工作代称；游戏内不把它解释成具体宗教体系中的神祇。它是执行轮回的不可见存在，不提供可战胜的实体形态。
- 完整游戏的核心身份真相在中后段揭示；垂直切片只确认主角属于这个家庭，并暗示他曾令家人恐惧。

## 目标受众与规模

- 目标受众：喜欢《Ib》《魔女之家》《狂父》等像素心理恐怖，以及知识驱动探索的成年玩家。
- 体验方式：单人、耳机优先、键盘操作。
- 垂直切片：12–15 分钟，两轮探索。
- 完整版目标：45–75 分钟，约五轮手工编排的叙事循环，12–16 个主要记忆节点。
- 首发目标：Windows、简体中文；后续平台和语言由测试反馈与维护能力决定。

## 明确不做

- 战斗、装备、技能树或资源刷取。
- 可以证明主角无罪或让惨剧未发生的“完美结局”。
- 依赖不可预测寻路的长时间追逐战。
- 没有叙事目的的程序化房间生成。
- 直接、猎奇地展示针对妻子或儿童的暴力过程。
- 为主播反应而设计的频繁跳脸或突然峰值音量。

## 产品成功的定义

成功不以“吓到所有人”为目标，而以是否产生了特定矛盾为准：玩家理解继续调查的危险，却仍因好奇、责任感或不甘心主动迈出下一步。

垂直切片的量化门槛见 `PLAYTEST.md`。完整版的外部评分、播放量和传播只作为结果指标，不能替代对核心体验的验证。
