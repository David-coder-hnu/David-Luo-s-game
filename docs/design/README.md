# 历史设计稿

本目录保存 2026-07-12 立项期间由规划工作流生成的原始材料。保留它们是为了追溯项目如何形成，不代表其中规格仍然有效。

已知被后续文档修正的内容包括：

- 单轮原型无法验证轮回后的心理变化，现改为可结束的双轮垂直切片；
- 不可见实体轴向追逐改为脚本化压迫段落；
- 垂直切片统一为不跨启动保存；
- 日记、地下室和第三轮不属于当前范围；
- 最后发现的任意碎片触发惩罚，不绑定结婚照；
- 随机程序化变异改为有叙事目的的手工变化集；
- 成功标准由开发者自我感受改为功能验收和不知情玩家观察。

当前文档入口为 [`../README.md`](../README.md)。任何实现、评审和测试都不得引用本目录作为需求来源。
