## Edge Cases
- 同一碎片重复交互 — 不应重复写入字典或重复弹出文本
- 玩家未收集全部碎片时无限探索 — 无时间限制，无强制事件
- 追逐/过场中退出游戏 — 下次启动从卧室重新开始，知识状态保留
- 碎片发现顺序 — 任意顺序都应该正常工作（非线性的知识解锁）
