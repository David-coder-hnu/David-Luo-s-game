# 音频设计与分镜

状态：**垂直切片声音基线已锁定**<br>
适用版本：Vertical Slice 0.1<br>
最后更新：2026-07-13

## 1. 声音命题

> 玩家不是被一声巨响吓到，而是在环境逐渐停止回应时意识到：另一个东西开始听见他了。

声音承担空间可信度、记忆距离和惩罚接近三种功能。沉默是可编排资源，不是缺少资产。

## 2. 原则

- 第一轮先建立可信住宅声景，再逐层拿走声音。
- 可定位声音属于房子；不可定位声音属于执行者。
- 不用突然峰值音量代替叙事铺垫。
- 关键线索即使不用耳机也能理解；耳机增加空间感，不提供独占答案。
- 每个声音必须说明来源、距离、叙事功能和是否真实发生。

## 3. 技术基线

- 源文件：48 kHz、24-bit WAV。
- 游戏内长循环：高质量 Ogg Vorbis；短事件可使用 WAV。
- 单声道用于明确位置的小物体，立体声用于房间底噪和非空间事件。
- 主输出 true peak 不高于 `-1 dBFS`。
- 常态声景建议落在 `-28` 至 `-22 LUFS` 的低强度范围；事件段落以可感知动态而非绝对大声为目标。
- 任一突发事件不应比此前十秒平均声景高出超过约 8 dB。
- 所有循环必须无点击衔接，并保留至少 50ms 淡入淡出控制空间。

数值是混音起点，最终以普通扬声器、封闭式耳机和低音量测试为准。

## 4. 音频总线

| 总线 | 内容 | 默认相对电平 | 处理 |
|---|---|---:|---|
| Master | 全部输出 | 0 dB | 峰值限制、最终计量 |
| Ambience | 水管、冰箱、房屋低频、房间音 | -12 dB | 轻微高切、循环管理 |
| Foley | 脚步、门、纸张、齿轮 | -8 dB | 按材质分组 |
| Memory | 字幕气息、餐具、钥匙、录音带 | -10 dB | 可带轻微带宽限制 |
| Event | 灯灭、木板受压、近距离呼吸 | -8 dB | 禁止过度压缩提高冲击 |
| Heartbeat | 惩罚心跳 | -18 dB 起 | 独立自动化，不做空间定位 |
| UI | 确认、取消、按住环 | -14 dB | 简短、无尖锐高频 |

主音量影响全部总线。切片不提供独立音效滑杆，以减少设置范围；开发混音面板可以单独静音总线。

## 5. 房间声景

### 卧室

- 远处水管每 18–28 秒一次，间隔使用固定序列而非真随机。
- 极低房屋结构声，避免明显旋律。
- 第二轮水管声比第一轮延迟 0.5 秒，作为潜意识重复差异。

### 厨房

- 冰箱压缩机低频循环，完成收据碎片后永久停止到本轮结束。
- 碎玻璃脚步不制作随机踩踏，避免玩家误以为存在伤害机制。
- 第二轮两只杯子只在首次进入时产生一次极轻陶瓷接触声。

### 儿童房

- 第一轮没有持续音乐盒，只有调查画纸时的纸张摩擦。
- 第二轮进入前可听到三音音乐盒；玩家越过触发线前 0.2 秒停止。
- 秦禾的哼唱只用于回避结尾，不在探索中反复消费。

### 客厅

- 第一轮时钟不走，没有滴答声。
- 第二轮错误时间产生一次机械回转，正确时间后完全静默三秒再开暗格。
- 录音带使用带宽受限的干燥质感，文字仍以字幕为准。

### 走廊

- 第一轮由远处木材热胀冷缩建立长度感。
- 惩罚时灯灭声从远到近，木板声不进行精确定位，让执行者无法被当成实体追踪。

## 6. 碎片声音语法

每个碎片遵循同一结构，但不能使用相同素材：

1. 物品声：纸张、蜡笔纸、相框木材；
2. 0.3–0.5 秒暗场；
3. 一段不完整记忆声音；
4. 回到房间后，Ambience 总线下降约 1.5 dB，并移除一个声源层；
5. 不播放“获得物品”提示音。

第三个任意碎片完成后不再恢复房间底噪，直接进入惩罚前静默。

## 7. 逐拍声音表

| 事件 | 声音 | 空间 | 时长 | 功能 |
|---|---|---|---:|---|
| 第一轮醒来 | 单次吸气、水管远响 | 吸气近场；水管左后 | 2.0s | 建立身体和住宅 |
| K 收据 | 橱柜门、女性气息“别出声” | 厨房近场 | 1.8s | 暗示躲避 |
| C 儿童画 | 数数到三、钥匙碰门 | 记忆单声道转右侧 | 2.4s | 建立回家恐惧 |
| L 结婚照 | 餐具轻碰、劝阻、长静默 | 客厅宽声场 | 3.2s | 确认家庭关系 |
| 惩罚 0–3s | 全部底噪淡出 | 全局 | 3.0s | 让玩家注意缺失 |
| 惩罚 3–7s | 心跳渐入、远灯灭 | 非空间 + 远端 | 4.0s | 建立接近 |
| 惩罚 7–12s | 门把、木板受压 | 近场但方向模糊 | 5.0s | 取消逃脱判断 |
| 惩罚 12–16s | 心跳放慢、空气变窄 | 非空间 | 4.0s | 反常而非加速 |
| 惩罚收束 | 近距离呼吸、2s 静默 | 监听者中心 | 3.0s | 完成惩罚 |
| 第二轮醒来 | 同一吸气、延迟水管 | 同开场 | 2.5s | 精确重复产生异常 |
| 时钟正确 | 卡扣、静默、暗格滑动 | 客厅墙内 | 3.8s | 给知识行动重量 |
| 面对结尾 | 磁带、撞门、停转 | 带宽受限 | 12–18s | 责任表达 |
| 回避结尾 | 三音哼唱 | 无方向 | 2.5s | 轮回承诺 |

## 8. 脚步

- 至少区分木地板与瓷砖两套，每套 4 个轻微变体。
- 变体按洗牌袋轮换，避免连续相同，也避免完全随机导致节奏突变。
- 步速与动画事件绑定，不由固定计时器独立播放。
- 惩罚移动减速时脚步同步放慢，不降调制造怪异喜剧效果。

## 9. 心跳

- 心跳不是生命值或敌人距离计量器。
- 起始接近安静，进入惩罚中段后清楚可闻。
- 最后四秒心率反而放慢，暗示结果已经确定。
- 不使用重低音增强插件把心跳推到身体不适。

## 10. 无障碍与测试

- 所有有语义的人声、低语和录音带内容必须有字幕。
- 扬声器测试中，玩家即使听不清方向也能完成全部谜题。
- 主音量 30% 时关键事件仍可感知，100% 时没有刺耳峰值。
- 单声道折叠后不得发生关键声音相消。
- 窗口失焦和暂停时所有渐变与循环暂停；恢复时不叠加第二份循环。

## 11. 命名与交付

命名格式：`category_room_event_variant.ext`，例如：

- `amb_kitchen_fridge_loop.ogg`
- `mem_child_counting_01.wav`
- `evt_punishment_breath_close.wav`
- `ui_hold_complete.wav`

每项资产同时提供来源工程、无损母版、游戏导入文件、循环点说明、响度/峰值记录和许可登记。

## 12. 音频完成标准

- 所有表内事件拥有可用素材或明确批准的静默。
- 三个碎片能只凭声音结构被辨认为不同记忆。
- 惩罚段落在五个房间触发时节奏一致且空间感合理。
- 普通扬声器、封闭式耳机、单声道各完整跑通一次。
- 没有事件依赖超过安全混音范围的音量制造效果。
- 发布包中不存在未登记来源的音频或字体。
